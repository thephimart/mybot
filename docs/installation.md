# Installation

## Requirements

- Python 3.11+
- pip

## Install

```bash
# Clone repository
git clone <mybot-repo>
cd mybot

# Create virtual environment (recommended)
python3 -m venv .venv
source .venv/bin/activate  # Linux/macOS
# .venv\Scripts\activate   # Windows

# Install in development mode
pip install -e ".[dev]"
```

## Verify Installation

```bash
mybot --version
```
